## Installation

```bash
$ npm install
```

## Set Up Environment Variables

Create a new file named `.env` in the root of your project and add the following content:

```env
DATABASE_URL=
ACCESS_TOKEN_KEY=
REFRESH_TOKEN_KEY=
MAIL_TRANSPORT=
MAIL_FROM=
JWT_SECRET=
AWS_REGION=
AWS_ACCESS_KEY_ID=
AWS_SECRET_ACCESS_KEY=
AWS_S3_BUCKET_NAME=
MOMO_YOUR_SECRET_KEY=
MOMO_PARTNER_CODE=
MOMO_ACCESS_KEY=
MOMO_REDIRECT_URL=
MOMO_IPN_URL=
```

## Running the app

```bash
# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```

Open http://localhost:3001/api with your favorite browser to swagger.
